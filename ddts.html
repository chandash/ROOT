Welcome to ClearDDTS info Page

===================================================================================================
   COMMAND                                                                                                         DESCRIPTION ===================================================================================================

ClearDDTS INSTALLATION 
inst    Install ClearDDTS and make the current machine the ClearDDTS server
dsbl    Disables (deinstalls) ClearDDTS.
mins    Modify installation parameters (such as ddts administrator names).
dbms    Build/rebuild the ClearDDTS database.
chdb    Change the RDBMS used by ClearDDTS.
REMOTE SITE CONNECTION
                
conn    Establish a connection between this site and another.
dcon    Remove the connection between this site and another.
CLASS & PROJECT CREATION/MAINTENANCE/DELETION
                 
clas    Create a new class.
meta    Create a new metaclass.
mmta    Modify metaclass parameters.
dcls    Delete a class or metaclass.
rcls    Rename a class or metaclass.

aprj    Add a new project to the system.
cprj    Close a project temporarily, preserving defect data.
dprj    Delete a project from this machine, reclaiming the disk space.
oprj    Reopen a closed project, or a project moved to this system.
mprj    Modify parameters of a project (e.g., mail notification lists).
renm    Rename a project.
bprj    Rebroadcast project information to the ClearDDTS network.
sprj    Save a project for transfer to another machine.
rprj    Restore a previously saved project on a new machine.
PROJECT SUBSCRIPTION
                    
asub    Subscribe to a project.
dsub    Delete a subscription to a project.
msub    Modify subscription parameters.
MAINTENANCE
smnt    Start maintenance mode.  Safely shuts system down for maintenance.
emnt    End maintenance mode.  Brings system back up.
LIST CONFIGURATION INFO
lprj    List all project parameters for specified project.
dsub    Delete a subscription to a project.
msub    Modify subscription parameters.
LIST CONFIGURATION INFO
lprj    List all project parameters for specified project.
lsit    List other hubs connected to this hub (via conn command).
ladm    List the ClearDDTS administrators on this machine.
lbug    List the projects that defects may be logged against.
lown    List the projects owned on this machine.
lsub    List the projects being subscribed to on this machine.
MISCELLANEOUS
alic    Add an evaluation license.
quit    Exit this program.
exit    Exit this program.
?       Give context sensitive help.
? cmnd  Give help on the commands above (e.g. ? aprj)
