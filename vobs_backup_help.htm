Telnet to ASAMAP2 (192.168.220.195)

Username : root

Password :

 

This machine has TWO network interface card. Make sure both of them are up and running.

 

Those 2 interfaces are,

 

bge0 : 192.168.220.195 Netmask : 255.255.255.0 (To external/public network)

hme0 : 192.178.16.21 Netmask : 255.255.0.0 (To reach VOB server (asamvob1))  

 

 

Bash prompt# ifconfig -a

lo0: flags=1000849<UP,LOOPBACK,RUNNING,MULTICAST,IPv4> mtu 8232 index 1

        inet 127.0.0.1 netmask ff000000

        inet 192.168.220.195 netmask ffffff00 broadcast 192.168.220.255

hme0: flags=1000843<UP,BROADCAST,RUNNING,MULTICAST,IPv4> mtu 1500 index 3
