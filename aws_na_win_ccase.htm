Checking the installation

Map any network drive to \\asamvob2\vobstore\AWS_NA_VOBS using the username cquser and password wipro123 and \\alcat56\udir\viewstore using your respective UNIX username and password. (Assuming views are located in /udir/viewstore partition of alcat56)

 

Open ClearCase Explorer (Start – programs – Rational ClearCase – ClearCase Explorer.)

Select Views from the General pane and right click and select “Add View Shortcut” as shown in below window.

 

 

Select View Type: as Dynamic and select any of the view Tag: (Here it is jeevan_na_view) and click OK

 

Select the view (i.e. jeevan_na_view) from General pane on the left side.

Right click the view and select “Mount VOB…” to mount your VOBs.

 

 

Select your VOB and click Mount which will mount the respective VOB as shown in below window.

 

Select the mounted VOB and try creating new text document to check whether user permission is available or not.(see below window)
