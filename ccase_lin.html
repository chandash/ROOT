Clearcase 5.0 in Redhat 9        Install clearcase in Linux by selecting almost all of the default settings from the release area  cd /usr/src  ln -s linux<version of linux> linux   cd /usr/src/linux/configs  cp kernel-2.4.20-i686.config /usr/src/linux/.config   cd /var/adm/atria/mvfs/  make   cd vnod_src  make  mount all the vobs in the /net/<vobserver>/  stop and start clearcase

Clearcase  2003.06.00 in RHEL 4 ES

cd /net/fluffy/Clearcase/2003.06.00/rhat_x86/clearcase/install/
./install_release

cd /var/adm/rational/clearcase/mvfs/vnode_src

Clean/purge old files:
make cleano

Create kernel config parameter file:
make vnode_param.mk.config

This creates file: vnode_param.mk.config
cat vnode_param.mk.config
RATL_EXTRAFLAGS := -DRHEL4
LINUX_KERNEL_DIR=/lib/modules/2.6.9-5.ELsmp/build
CONFIG_MVFS=y
CONFIG_MVFS_VNODE=m

make
make install

ls -l /lib/modules/2.6.9-5.ELsmp/kernel/fs/mvfs/

-rwxr-xr-x 1 root root 1406550 May 8 16:12 vnode.ko

/etc/init.d/clearcase stop
/etc/init.d/clearcase start
