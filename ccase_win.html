  1. Create group name auto ( Rht click My Computer à manage à Local users and groups à groups (Rht Click) à New Group)

 

 

  1. Add your windows username to that auto group .

 

  1. Create new User Variable by Right Clicking

 

 My computer -> Properties -> Advanced -> Environment Variables.,

 

  1. Variable name as CLEARCASE_PRIMARY_GROUP and Variable value as auto as shown in the following window.

 

  1. Similarly create System Variable with same values.,

 

  1. After creating these variables, Re-login (logout and login), to make active these system variables.,
