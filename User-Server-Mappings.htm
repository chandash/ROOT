FTTU server and User Mappings
Note:

1. Developers!!! If you have done any changes in a checked-out element please take copy or backup of that file (checked-out element) in your home directory. This recommendation is to prevent data loss during harddisk crash or server down.

2. You can still create and delete views on FTTU and FTTU-BUILD servers.
    E.g:
    To create a view jeevan_test on viewstore partition of FTTU from fttu42 server will be
          ct mkview -tag jeevan_test -host fttu42 -hpath /net/fttu/viewstore/jeevan_test.vws -gpath /net/fttu/viewstore/jeevan_test.vws /net/fttu/viewstore/jeevan_test.vws

     To delete remotely created view jeevan_test will be
         ct rmview /net/fttu/viewstore/jeevan_test.vws
FAQs:

1. How to access incoming and outgoing directory of FTTU if I'm not using FTTU server??
    Solution: cd /net/fttu/sfw

2. If my dedicated machine goes down which machine should I need to use??
    Solution: You have to contact Alcatel-Support team along with the details of the machine which went down and the users list who are all affected. They will provide alternate 
                  machine to the affected engineers.
